# Popular Packages by Category

| Package | Description |
|---------|-------------|
| `pi-subagents` | Delegate tasks to subagents with chains, parallel execution |
| `pi-interactive-shell` | Run AI agents as foreground subagents in TUI overlays |
| `pi-finder-subagent` | Read-only local workspace scout subagent |
| `@tmustier/pi-agent-teams` | Claude Code agent teams style workflow |

## Install Commands

```bash
# Install any package
pi install npm:<package-name>

# Try without installing
pi -e npm:<package-name>

# Project-local
pi install npm:<package-name> -l
```
